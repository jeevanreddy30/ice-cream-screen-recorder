+-[ HOW TO USE ]---------------------------------------------------------------+
|                                                                              |
| Interactive Mode (Default)                                                   |
| ==========================                                                   |
|                                                                              |
| Start the activator, select the product to active from the list and click    |
| "Activate".                                                                  |
|                                                                              |
| Silent or Very Silent Mode                                                   |
| ==========================                                                   |
|                                                                              |
| Use the command line options described in "Command Line Parameters/Exit      |
| Codes" section.                                                              |
|                                                                              |
+-[ COMMAND LINE PARAMETERS | EXIT CODES ]-------------------------------------+
|                                                                              |
| Options (use "/" or "-")                                                     |
| ========================                                                     |
|                                                                              |
| /activate:<index>                                                            |
|                                                                              |
| Performs a silent activation of the product indicated by its index value.    |
| Allowed values for product index are:                                        |
|                                                                              |
| 0 - Icecream Ebook Reader                                                    |
| 1 - Icecream Image Resizer                                                   |
| 2 - Icecream PDF Candy Desktop                                               |
| 3 - Icecream PDF Converter                                                   |
| 4 - Icecream PDF Split and Merge                                             |
| 5 - Icecream Screen Recorder                                                 |
| 6 - Icecream Slideshow Maker                                                 |
|                                                                              |
| /help or /?                                                                  |
|                                                                              |
| Shows a message box with a short description of the standard command line    |
| parameters.                                                                  |
|                                                                              |
| /overwrite                                                                   |
|                                                                              |
| Overwrite an existing license file. If this option is not used and a license |
| file already exists, the activator will ask for overwrite confirmation only  |
| if the "/verysilent" option was not used, otherwise it will abort the        |
| activation.                                                                  |
|                                                                              |
| /verysilent                                                                  |
|                                                                              |
| Performs a very silent activation. It does not show any dialog or message    |
| box.                                                                         |
|                                                                              |
| Exit Codes                                                                   |
| ==========                                                                   |
|                                                                              |
| 0: No error (nothing done).                                                  |
|                                                                              |
| 1: Integrity error, activator executable was modified or is corrupted.       |
|                                                                              |
| 2: Activator is already running.                                             |
|                                                                              |
| 3: Administrative privileges are required (actually not used, the activator  |
| already requests for elevation when executed).                               |
|                                                                              |
| 4: A fatal error has occurred and cannot continue.                           |
|                                                                              |
| 5: Activation succeeded.                                                     |
|                                                                              |
| 6: Activation aborted.                                                       |
|                                                                              |
| 7: Activation failed.                                                        |
|                                                                              |
| 8: Invalid product index specified.                                          |
